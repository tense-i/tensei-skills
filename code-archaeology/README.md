## 描述

以解决方案架构师视角，基于 Git 历史（log/diff/blame/PR）对代码仓库进行"考古式"演进分析。从提交记录、PR 历程、文件变更中反推系统的演进阶段、架构变迁、关键决策与方案迭代轨迹，输出超长演进分析文档（>1000行）。不需要 checkout 旧版本。
