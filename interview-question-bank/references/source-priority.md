# Source Priority and Search Heuristics

## Best sources for current hiring demand
1. Official company career pages
2. Official campus recruiting pages
3. Nowcoder job listings and interview posts
4. Recruitment platforms (Boss, Zhilian, 51job, Lagou, Liepin, Haitou)

## Best sources for real questions
1. Nowcoder interview experience / written test collections
2. Public interview recap posts with company, role, and date context
3. Public technical communities that clearly describe interview prompts

## Heuristics
- When a topic is broad, search both role-level and technology-level queries.
- Use at least one minimally rewritten search query close to the user's wording.
- Prefer public sources with recent dates for market demand.
- Use older technical references only to explain stable fundamentals, not market trends.
- When multiple sources repeat the same theme, treat it as a signal for “热门题”.
