# Post Deep-Dive Analyst（帖子深度解读分析器）

Auto-trigger keywords: 帖子分析, 分析这个帖子, 深度解读, 观点分析, 评论分析, 认知模型, 方法论提炼, 短内容扩展, 社交媒体分析, post analysis, deep dive, 知乎分析, 小红书分析, 推文分析, 微博分析, 即刻分析, 观点挖掘, 洞察提炼, 帖子解读, 评论区分析

## Purpose

将社交媒体帖子（X/知乎/小红书/即刻/微博等）的正文与评论区内容，深度分析并扩展为一份**信息量极高的长文档**。

## Core Capabilities

- 从短文本抽取"观点/论证/隐含前提/价值取向/方法线索"
- 还原作者的认知模型与看问题的独特视角
- 做批判性分析（反例、边界、风险、可证伪点）
- 逆向工程：把零散洞察还原成完整方法论
- 评论区价值挖掘与争议辨析
- **必须检索资料**配合输入内容做深度扩展
- 输出一份"可转发、可教学、可落地"的结构化长文档

## Usage

提供帖子正文和评论区内容，要求 Claude 进行深度分析：

```
分析这个帖子：[粘贴正文]

评论区：[粘贴评论]
```

## Input

- **source_text**（必填）：帖子正文
- **comments**（强烈建议）：评论区内容
- **platform**：来源平台（X/知乎/小红书/即刻/微博等）
- **domain**：领域
- **author_info**：作者身份背景

## Output

一份结构化的深度分析长文档（Markdown），包含：
1. 快速结论（核心观点 + 一句话摘要）
2. 作者观点解码（来源推断/隐含假设/认知模型）
3. 深度扩展（逐条观点多维度展开）
4. 评论区深度分析（情绪图谱/高价值评论/争议焦点）
5. 批判性分析（反例/边界/风险/可证伪信号）
6. 方法论逆向工程（决策树/检查清单/流程）
7. 落地行动清单
