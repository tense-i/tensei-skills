# Short Insight Deep Dive Output Template

## A. 文档头部

1. 标题 + 一句话摘要
2. 原文核心观点列表
3. 适用读者与阅读收益

## B. 作者观点解码

1. 观点来源推断
2. 作者隐含假设
3. 前置知识清单
4. 认知模型还原

## C. 深度扩展

逐条对应原文观点，不合并偷懒。

对每条观点至少补以下 4 类：
- 定义
- 机制
- 条件
- 反例
- 指标
- 案例
- 操作

## D. 批判性分析

- 逻辑漏洞 / 样本偏差 / 适用边界
- 与主流观点的冲突点
- 风险：误用会导致什么
- 可证伪信号

## E. 落地与迁移

1. 行动清单：最小可行行动 / 进阶行动 / 团队级行动
2. 常见误区与 FAQ
3. 延伸阅读 / 相关模型

## F. 逆向工程结构

至少输出两种可执行结构，例如：
- 决策树
- 检查清单
- 指标体系
- 输入 -> 判断 -> 行动 -> 验证 -> 复盘流程
